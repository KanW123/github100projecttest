# シナリオ案001: 終わらない夜の九龍城

作成日: 2026-01-27

## コンセプト

**ループもの × 九龍城の日常**

主人公は特別な人間じゃない。ただ九龍城に迷い込んで、気づいたら「同じ夜」を繰り返している。
住人たちにとって「時間の乱れ」は日常茶飯事。慣れている。

**キーワード**: 終わらない夜、ループ、日常の中の異常、選択

---

## 導入

### オープニング

主人公は夜の九龍城に迷い込む。理由は曖昧でいい（観光で迷った、噂を聞いて来た、気づいたらいた）。

狭い路地、蒸気、ネオン。異様だが活気のある街。
バーで一杯飲んで、魚屋を冷やかして、階段を上って…一晩を過ごす。

朝になって外に出ようとすると——また同じ夜の始まりに戻っている。

### 最初のループ

混乱する主人公。でも住人たちは平然。

**バーのマスター（トゲトゲ）**:
「ああ、アンタも引っかかったか。よくあることだ」
「出口？ あるにはあるが、簡単じゃない」
「まあ焦るな。ここじゃ時間はいくらでもある」

この街では「時間の乱れ」は天気のようなもの。
雨が降ったら傘をさす。ループしたらまた同じ一日を過ごす。それだけ。

## ゲーム構造

### 基本サイクル

```
夜の始まり（廊下・スタート地点）
    ↓
探索（各エリアを回る、住人と会話）
    ↓
イベント発生（特定条件で）
    ↓
夜の終わり → ループ
```

### ループの進行

| 周回 | 状態 |
|------|------|
| 1周目 | 手探り。誰も詳しく教えてくれない。基本情報だけ |
| 2-3周目 | 住人が「またか」と認識。少しずつ情報が増える |
| 4-5周目 | 信頼関係ができる。深い話を聞ける |
| 6周目以降 | 核心に近づく。「心臓」の存在を知る |

### フラグ管理

- **会話フラグ**: 誰と何回話したか
- **信頼度**: 住人ごとの親密度（ループを重ねると上がる）
- **アイテム**: ループしても一部は持ち越せる（「時間の欠片」など特殊アイテム）
- **知識フラグ**: 一度知った情報は次周に引き継がれる

## キャラクター詳細

### 既存キャラ（変更なし）

#### バーのマスター
- **名前**: 未定（住人は「マスター」と呼ぶ）
- **ビジュアル**: トゲトゲした見た目、水槽がある
- **役割**: 最初の案内役。ループについて教えてくれる
- **性格**: 飄々としている。深入りしない主義

#### 魚屋
- **名前**: 未定
- **ビジュアル**: 既存素材
- **役割**: 日常の象徴。ループしても変わらない日常
- **特徴**: 毎回同じセリフで同じ魚を勧める → 何周もすると微妙に変化？

### 新規キャラ

#### 阿婆（アポ）
- **場所**: 階段上の先
- **ビジュアル案**: 小柄な老婆。背中から細い蒸気パイプが何本も。片目が時計の文字盤になっている
- **設定**: 九龍城で最も長く生きている（何百ループも経験）。「心臓」の場所を知っている
- **性格**: 気難しい。信頼を得るまで何も教えない
- **口癖**: 「急いでどうする。ここじゃ時間だけは腐るほどある」

#### 肥仔（フェイジャイ）
- **場所**: 修理屋
- **ビジュアル案**: 巨漢。両腕が肘から先が機械（複数のツールが内蔵）。顔は人懐っこい
- **設定**: 何でも直せる修理屋。自分の腕も自分で作った
- **性格**: 陽気。話好き。でも過去の話になると黙る
- **役割**: アイテム改造、機械系イベントの鍵

#### 細龍（サイロン）
- **場所**: 路地裏
- **ビジュアル案**: 痩せぎす。顔の半分に歯車模様の刺青。目が異様に鋭い
- **設定**: 裏情報屋。この街で起きることは大体知っている
- **性格**: ビジネスライク。金か「時間の欠片」がないと口を開かない
- **役割**: 有料情報提供。ショートカット的な存在

## メインストーリーライン

### 第一幕: ループの日常

**目標**: ループに慣れる。住人と顔見知りになる

- ループの仕組みを理解する
- 各エリアを探索、住人と会話
- 「出口」があるらしいと知る
- 阿婆の存在を聞く（でもまだ会えない）

**イベント例**:
- バーで他の「迷い込んだ人」の話を聞く（過去にもいた）
- 魚屋で「この魚、昨日も売ってましたよね」→「昨日？何言ってんだ今日入荷だぞ」
- 階段上で阿婆を見かけるが、話しかけても無視される

### 第二幕: 深層へ

**目標**: 信頼を得る。「心臓」の存在を知る

- 阿婆と話せるようになる（条件: 複数周回 + 特定アイテム）
- 「時計仕掛けの心臓」の存在を知る
- なぜループが起きているのか、断片的に理解
- 細龍から「心臓」への道の情報を買う

**イベント例**:
- 肥仔に「時間の欠片」を見せると、それで動く機械を見せてくれる
- 阿婆が昔話をする:「この街はな、本当はもうないんだよ」
- 細龍が警告:「心臓に近づくな。あれに触れた奴は皆、消えた」

### 第三幕: 選択

**目標**: 「心臓」にたどり着く。最終選択

- 蒸気塔→地下への道が開く
- 「心臓」と対面
- 真実を知る: 九龍城は外の世界では既に取り壊されている。心臓がループを起こすことで、この街と住人は存在し続けている
- 選択を迫られる

## 真実と選択

### 真実

「時計仕掛けの心臓」は九龍城の中心で、時間を巻き戻し続けている。

**なぜ？**
- 外の世界では九龍城はとっくに取り壊された（1990年代）
- 誰かが「心臓」を作り、九龍城を時間の中に閉じ込めた
- ループすることで、街も住人も「存在」し続けている
- 心臓を止めれば、九龍城は「本来の時間」に戻る＝消滅する

**住人たちは知っているのか？**
- 古参（阿婆など）は知っている
- 若い住人は知らない or 気づかないふりをしている
- マスターは知っているが、どうでもいいと思っている

### 最終選択

| 選択肢 | 内容 | 結果 |
|--------|------|------|
| **心臓を止める** | ループを終わらせる | 主人公は外に出られる。九龍城は消滅。住人たちは？ |
| **心臓を守る** | このまま続ける | 主人公は住人として永遠にここで暮らす。ループは続く |
| **心臓を作り変える** | 第三の道（条件厳しい） | ループを緩やかに。街は残り、主人公も出られる。でも代償がある |

## エンディング案

### ED1: 夜明け（心臓を止める）
主人公が外に出ると、そこは現代の香港。九龍城があった場所は公園になっている。
振り返っても、もう入口はない。

ただ、ポケットには「時間の欠片」が一つ残っている。
それを握ると、かすかに——バーのマスターの声が聞こえる気がする。

### ED2: 永遠の住人（心臓を守る）
主人公はもう外に出ようとしない。
バーで一杯。魚屋で魚を買う。階段を上って阿婆と話す。
同じ夜が繰り返される。でも、悪くない。

「おう、また来たか」
マスターがいつもの席にグラスを置く。
永遠の夜が、今日も始まる。

### ED3: 新しい心臓（心臓を作り変える）
※条件: 全キャラの信頼度MAX + 特定アイテム全収集

肥仔と協力して、心臓を「改造」する。
完全なループではなく、緩やかな時間の流れ。
街は残る。住人も残る。でも、少しずつ変わっていける。

主人公は「出入り自由」になる。
外と中を行き来できる、唯一の存在として。

### ED4: 消失（バッドエンド）
※条件: 心臓に触れる際に特定の選択を誤る

心臓が暴走。時間が崩壊する。
主人公も、街も、住人も、全てが「なかったこと」になる。

## 演出アイディア

### ループ表現
- 夜の始まりは必ず「廊下」から
- 同じBGM、同じ環境音
- でも周回ごとに微妙な変化（最初は気づかない）

### 時間の乱れ表現
- たまに画面がノイズ走る
- 同じセリフが二重に聞こえる
- 時計が逆回転している背景小物

### 住人の変化
- 1周目: 他人行儀
- 5周目: 「お前もだいぶ長いな」
- 10周目: 「もう仲間みたいなもんだ」
